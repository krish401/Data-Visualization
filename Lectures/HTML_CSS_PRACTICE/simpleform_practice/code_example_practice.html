Course Objectives

In this course we will learn how the internet works and how we can create webpages. We will use HTML, CSS and basic JavaScript in this course for designing the webpages. As we might have students in this course with no previous programming experience, therefore, the pace of the course is designed for beginners. We will not go into any server side scripting and the class discussions will be limited to the client side programming. Assignments will augment the class lectures. To ensure the development of the necessary competencies, assigned homework includes the development of program solutions to problems of adequate complexity and relevance.
